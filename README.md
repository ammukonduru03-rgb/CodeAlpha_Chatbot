# 🤖 Interactive Rule-Based Chatbot

An interactive GUI-based chatbot developed using Python and Tkinter as part of the **CodeAlpha Python Programming Internship**.

This chatbot responds to user inputs using predefined rules and provides an engaging conversational experience through a graphical user interface.

---

## 📌 Project Overview

The goal of this project was to understand the fundamentals of chatbot development using Python while implementing a user-friendly graphical interface.

The chatbot can handle greetings, answer predefined questions, display the current date and time, provide motivational quotes, tell jokes, and assist users through a simple help menu.

---

## ✨ Features

- Interactive GUI built using Tkinter
- Custom chatbot logo integration
- Rule-based conversation handling
- User name recognition
- Date and time display
- Motivational quotes
- Fun joke responses
- Help command support
- User-friendly chat interface
- Input handling with punctuation support

## 🚀 How to Run the Project

### 1. Clone the Repository

```bash
git clone https://github.com/ammukonduru03-rgb/CodeAlpha_Chatbot.git
```

### 2. Navigate to the Project Folder

```bash
cd CodeAlpha_Chatbot
```

### 3. Install Required Package

```bash
pip install pillow
```

### 4. Run the Application

```bash
python chatbot_gui.py
```
